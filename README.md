# Website Bán Hàng Thời Trang STM

## Giới thiệu
STM là dự án website bán hàng thời trang được xây dựng theo hướng fullstack, phục vụ cho học tập, phát triển kỹ năng và làm portfolio cá nhân.  
Dự án mô phỏng một hệ thống bán hàng thực tế với giao diện người dùng, quản trị admin, xử lý đơn hàng và tích hợp thanh toán.

## Chức năng chính
- Hiển thị danh sách sản phẩm theo danh mục
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Thêm vào giỏ hàng và đặt hàng
- Đăng ký / đăng nhập tài khoản
- Quản lý đơn hàng cá nhân
- Trang quản trị sản phẩm, đơn hàng và dashboard
- Tích hợp thanh toán MoMo

## Công nghệ sử dụng
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Khác:** Git, GitHub, MoMo Payment

## Cấu trúc dự án
- `client/`: giao diện người dùng
- `server/`: backend xử lý API và nghiệp vụ
- `database/`: file cấu trúc và dữ liệu SQL

## Điểm nổi bật
- Xây dựng website bán hàng fullstack hoàn chỉnh
- Có cả giao diện người dùng và quản trị viên
- Kết nối cơ sở dữ liệu PostgreSQL
- Tích hợp thanh toán MoMo
- Phù hợp làm đồ án và portfolio cá nhân

## Hướng phát triển
- Hoàn thiện giao diện responsive
- Tối ưu trải nghiệm người dùng
- Bổ sung kiểm thử và bảo mật hệ thống
- Triển khai demo online trong tương lai
